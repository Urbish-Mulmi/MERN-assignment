NOTES:
        Theres two type of input categorization:
        1)Open input: Users can give unique values

          eg.input type=[text, tel, password, email]
          here attributes used are:
            label for, input type, id, name,placeholder
          in backend values are passed as:
            key:value ; 
            using attributes:-
              name: value 
              (name is defined in input attribute and
               value is passed via user entered uniq value)  

        2)Closed input: Users are to choose a value from displayed options

          eg.checkbox, radio, select(dropdown)
          here attributes used are:
           label for, input type,select tag, id, name, value
          in backend values are passed as:
            key:value ; 
            using attributes:-
              name: value (both are attributes defined along input tag)
